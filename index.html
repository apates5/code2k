Code2k

A programming language for simulating 3 point contests.

GitHub Repository

Code2k is a domain-specific programming language designed for simulating customizable 3-point basketball contests. Easily create players, set up contests, and run fun or competitive games â€” including creative challenges like FizzBuzz modes.

Key Features

  • Easy-to-read English-like syntax for player and game setup
  • Support for different player forms (Fast, Smooth, Slow)
  • Multiple difficulty levels (Rookie to Hall of Fame)
  • Optional FizzBuzz mode
  • Realistic shot success based on ratings and difficulty

Sample Program

player Coby_White {
    rating: 90
    form: Smooth
}

player Curry {
    rating: 97
    form: Fast
}

game {
    shots: 25
    difficulty: Superstar
    players: [Coby_White, Curry]
    fizzbuzz: false
}

Output

3-Point Contest (Difficulty: Superstar, Shots: 25)
Coby_White (Smooth): 17/25 shots made
Curry (Fast): 22/25 shots made

Winner(s): Curry
